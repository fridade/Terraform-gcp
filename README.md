# Terraform Infrastructure Setup

## Overview

This README provides an overview of the Terraform configuration used to manage various Google Cloud Platform (GCP) resources. It covers the setup of a remote backend using Google Cloud Storage (GCS), as well as resources like Cloud SQL, Cloud Run services, load balancers, and Google Cloud Storage buckets. Each section includes explanations and relevant links to GCP documentation.
